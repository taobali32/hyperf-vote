# 包作用
简化你的点赞/取消点赞 收藏/取消收藏。

# 安装
```
composer require "jtar/hyperf-vote"

php bin/hyperf.php vendor:publish jtar/hyperf-vote

# 执行迁移(可能会破坏你的迁移文件,谨慎操作)
php bin/hyperf.php migrate

```
